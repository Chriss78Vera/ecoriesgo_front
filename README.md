# EcoRiesgo Front

Aplicacion web para evaluar riesgos ambientales por zona y visualizar resultados en mapa y dashboard.

## Autores

- Christopher Vera
- Jordan Paillacho
- Lenin Simaliza

## Requisitos

- Node.js
- pnpm

## Configuracion

El proyecto usa la variable de entorno:

```env
BACK_ECORIESGO=
```

## Ejecucion

Instalar dependencias:

```bash
pnpm install
```

Iniciar servidor de desarrollo:

```bash
pnpm dev
```

Generar build:

```bash
pnpm build
```
